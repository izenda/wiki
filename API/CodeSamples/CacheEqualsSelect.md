#CacheEqualsSelect

[[_TOC_]]

##About

Gets or sets the value indicating whether the "Equals (Select)" value should be cached. This will cause queries that are used to populate Equals (Select) lists to re-use their results and not be re-executed on the database.

_**Note:** If two different fields in two different tables yield the same results, Izenda will see two different queries and need to cache the results of each query respectively._

**Default Value:** False

      return;
    AdHocSettings.LicenseKey = "INSERT_LICENSE_KEY_HERE";
    AdHocSettings.SqlServerConnectionString = "INSERT_CONNECTION_STRING_HERE";
    Izenda.AdHoc.AdHocSettings.AdHocConfig = new CustomAdHocConfig();
    AdHocSettings.CacheEqualsSelect = true; //The relevant setting
    HttpContext.Current.Session["ReportingInitialized"] = true;
  }
}
