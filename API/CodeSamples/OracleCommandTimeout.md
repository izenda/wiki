#OracleCommandTimeout

[[_TOC_]]

##About

_**Obsolete:** Unlike the Command object in the other .NET Framework data providers (SQL Server, OLE DB, and ODBC), the OracleCommand object does not support a CommandTimeout property. Setting a command timeout has no effect and the value returned is always zero._

Sets the time Izenda Reports will wait for an Oracle command to execute before timing out.

**Default Value:** null
