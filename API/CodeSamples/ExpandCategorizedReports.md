#ExpandCategorizedReports

[[_TOC_]]

##About

Gets or sets whether sub-categories will be expanded by default on the Report List. This setting applies to both the old (classic) and new (AJAX-enabled) versions of the report list.

**Default Value:** true

    AdHocSettings.ExpandCategorizedReports = false; //The relevant setting
    AdHocContext.Initialized = true;
  }
}
