#ApplyAntiXssToReportOutput

[[_TOC_]]

##About

When this option is enabled, report headers, footers, and other free-form input areas that accept HTML will be encoded when sent to the server and displayed to the user. When disabled, the HTML will be interpreted as executable markup an any script tags will be run. Users would be able to design headers and footers with dynamic parts and interactivity with this disabled.

**Default Value:** false
