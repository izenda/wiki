#ExtendedFormats

[[_TOC_]]

##About

_**Obsolete:** Use Formats instead._

Gets or sets the collection of additional formats that will be displayed in the ReportDesigner's "Format" dropdown lists.

**Default Value:** null

    AdHocContext.Initialized = true;
  }
}
