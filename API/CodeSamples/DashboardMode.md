#DashboardMode

[[_TOC_]]

##About

_**Obsolete:** Use Izenda.Web.UI.DashboardDesigner instead._

Gets or sets whether dashboard mode is enabled. Dashboard mode is an alternative view on the Report Designer page that has nothing to do with actual dashboards.

**Default Value:** false

    AdHocContext.Initialized = true;
  }
}
