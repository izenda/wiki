#AutoOrForRepeatFilters

[[_TOC_]]

##About

Determines whether OR operator is used to concatenate two subsequent filters that use the same column. Although it is a better practice to use the filter logic feature, enabling this option can let users select the same column multiple times and each filter will automatically apply the OR operator between the values of each subsequent filter with the same column.

**Default Value:** false
