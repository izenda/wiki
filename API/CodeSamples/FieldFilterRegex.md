#FieldFilterRegex

[[_TOC_]]

##About

_**Obsolete:** Use FiltersRegex instead._

Gets or sets the regular expression used for column filtering. If name of a column matches this expression then it will not be shown.

**Default Value:** MD5|^(Passw)
