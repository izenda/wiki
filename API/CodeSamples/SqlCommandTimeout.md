#SqlCommandTimeout

[[_TOC_]]

##About

Gets or sets the wait time in seconds for dropdown menus that use results loaded from database queries. If the wait time has elapsed and the query has not returned, the command ends.

**Default Value:** 3600

      return;
    AdHocSettings.LicenseKey = "INSERT_LICENSE_KEY_HERE";
    AdHocSettings.SqlServerConnectionString = "INSERT_CONNECTION_STRING_HERE";
    Izenda.AdHoc.AdHocSettings.AdHocConfig = new CustomAdHocConfig();
    AdHocSettings.SqlCommandTimeout = 60; //The relevant setting
    HttpContext.Current.Session["ReportingInitialized"] = true;
  }
}
