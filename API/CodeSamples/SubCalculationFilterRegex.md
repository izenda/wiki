#SubCalculationFilterRegex

[[_TOC_]]

##About

Gets or sets a regular expression which determines which fields should be excluded from subtotal calculations by default. This is determined by characters present in the literal field name.

**Default Value:** 

Before version 6.10.0.35: ID$|Id$|^id$|_id$

After version 6.10.0.35: blank
