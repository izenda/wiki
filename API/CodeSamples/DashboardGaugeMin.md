#DashboardGaugeMin

[[_TOC_]]

##About

_**Obsolete:** As of Izenda 6, this setting is not recommended for use. Dashboard gauges always use the MaxValue for the value limit._

Gets or sets the guage min value for the "Dashboard Gauge" format.

**Default Value:** -1.79769313486232E+308 (double.MinValue)

  }
}
